# Сводка: Кривые обучения топ-5 моделей

## Задача
Создать ноутбук для генерации графиков кривых обучения для топ-5 моделей по награде.

## Топ-5 моделей по награде (best_eval_reward)
1. **ppo_seed42_5000K**: 294.04 ± 21.93 (5M шагов)
2. **ppo_seed42_1000K**: 268.10 ± 12.26 (1M шагов)
3. **ppo_seed42**: 255.72 ± 27.06 (500K шагов)
4. **ppo_seed42_400K**: 248.55 ± 18.60 (400K шагов)
5. **ppo_seed42_500K**: 244.57 ± 21.89 (500K шагов)

## Созданные файлы

### 1. Python скрипт (standalone)
- **Файл**: `notebooks/top5_learning_curves.py`
- **Описание**: Исполняемый скрипт для генерации графиков
- **Запуск**: `python notebooks/top5_learning_curves.py`

### 2. Jupyter Notebook (интерактивный)
- **Файл**: `notebooks/top5_learning_curves.ipynb`
- **Описание**: Интерактивный ноутбук для исследования данных
- **Содержит 6 ячеек**:
  1. Заголовок и описание
  2. Импорт библиотек
  3. Определение путей к моделям
  4. Загрузка данных метрик
  5. График 1: Награда vs Шаги
  6. График 2: Награда vs Эпизоды
  7. Сводка

### 3. Сгенерированные графики
- **Файл**: `results/reports/top5_reward_vs_timestep.png` (263 KB)
  - Оси X: Шаги обучения (timesteps)
  - Оси Y: Награда (Reward)
  - Подписи на русском языке
  - Легенда с названиями экспериментов
  - Разные цвета для каждой модели

- **Файл**: `results/reports/top5_reward_vs_episode.png` (254 KB)
  - Оси X: Эпизоды
  - Оси Y: Награда (Reward)
  - Подписи на русском языке
  - Легенда с названиями экспериментов
  - Разные цвета для каждой модели
